## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ivanlin0418/ivanlinai.git
   ```

2. Navigate to the project directory:

   ```bash
   cd react-portfolio
   ```

3. Install the dependencies:

   ```bash
   npm init
   npm install
   ```

## Usage

To start the development server, run:

```bash
npm run dev
```
